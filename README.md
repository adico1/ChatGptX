# ChatGptX Chrome Extension

## Introduction
ChatGptX is a Chrome Extension that enables developers to generate code snippets and templates using natural language. The extension integrates with the OpenAI GPT-3 language model to provide accurate and relevant code suggestions to users.

## Installation
To install ChatGptX, follow these steps:
1. Clone the repository from GitHub.
2. Open the Chrome browser and navigate to the extensions page.
3. Enable developer mode.
4. Click on "Load unpacked" and select the "build" folder in the cloned repository.

## Features
- Generate code snippets and templates using natural language.
- Save and organize snippets and templates for easy access.
- Share snippets and templates between chats.
- Search for snippets and templates using tags.
- Create and modify your own templates.
- Secure and private storage of data using local storage.
- Export and import saved snippets and templates.

## Usage
To use ChatGptX, follow these steps:
1. Open the ChatGptX extension.
2. Initiate a chat with the OpenAI GPT-3 language model.
3. Type in your natural language query for a code snippet or template.
4. Choose the desired snippet or template from the suggestions provided.
5. Paste the selected snippet or template into your code editor.
